# Library Catalog Tool

## Overview

A Java console-based application that helps manage library books efficiently.

## Features

* Add Book
* View Books
* Search Book
* Store Book Title and Author Details
* User-Friendly Menu System

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* ArrayList
* Scanner Class

## Learning Outcomes

* Classes and Objects
* ArrayList Operations
* User Input Handling
* Search Functionality
* Java Programming Fundamentals
